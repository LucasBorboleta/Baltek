Playing modes

Hereafter are the playing modes:

  • The two local players share the same Web browser.
  • Each of the two remote players uses his/her own Web browser.
  • The human player plays against an artifitial intelligence (AI).
  • Two AI play against each other.

Only the mode two local players is implemented even if the bouton New game is coming together with the selector Human/AI ....

Rules

Access the rules via the button Rules.

Portability

BALTEK is played within a Web browser, and so is relatively portable. However, its current ergonomy requires a big enough screen and a mouse.

The adaptation of BALTEK to smartphone is foreseen in a next phase.
