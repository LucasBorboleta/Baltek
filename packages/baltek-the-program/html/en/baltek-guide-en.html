Settings

is used to select the playing mode, language, etc.

Here are the playing modes:

  • The two close players share the same Web browser.
  • Each of the two remote players uses his/her own Web browser.
  • The human player plays against an artificial intelligence (AI).
  • Two AI play against each other.

Modes with AI are not yet implemented.

Invitation

shows or hides the TogetherJS interface that allows you to invite a remote friend to play BALTEK.

The TogetherJS interface allows two remote players to synchronize their web browsers and also to chat.

The TogetherJS interface provides help in English. Click on the avatar.

Synchronize the two web browsers before starting a new game.

Both players must apply themselves the rule of alternating turns, because the passive player's web browser is never blocked.

Rules

Portability

BALTEK is played within a Web browser, and so is relatively portable. However, its current ergonomics requires a big enough screen and a mouse.

The adaptation of BALTEK to smartphone is foreseen in another phase.
