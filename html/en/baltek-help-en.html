Interface

Hereafter are a few explanations on how to play moves:

  • display the scores and the availability of the sprint bonuses for the two players.
  • displays the credits of the active player.
  • confirms or cancels/reinitializes the turn of the active player.
  • selects the sprint bonus.
  • Unselect the selected footballer/ball prior to go on for a new selection.

Playing modes

Here are the playing modes:

  • The two close players share the same Web browser.
  • Each of the two remote players uses his/her own Web browser.
  • The human player plays against an artificial intelligence (AI).
  • Two AI play against each other.

Only the two close players mode is currently implemented even if the Human / IA ... selector is displayed next to the New Game button.

Rules

Access the rules via the button Rules.

Portability

BALTEK is played within a Web browser, and so is relatively portable. However, its current ergonomics requires a big enough screen and a mouse.

The adaptation of BALTEK to smartphone is foreseen in a next phase.
